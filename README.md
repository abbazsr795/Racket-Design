# RacketDesign (Free React Components from Racket Labs)

## In Action
https://racket-design.vercel.app/

## Prerequisites
* Zustand (or any state management tool)
* Tailwind CSS
* Evergreen-UI

## Components
* SignIn Box - "/signin"
* SignUp Box - "/signup"
* Settings Box - "/settings"
* Header - Every dir
* Footer - Every dir
* Buttons - "/buttons"
* Cards - "/cards"
